# Dataset
Stroke Prediction Dataset - https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset?resource=download

## Download Data
- Visit the link above and click download in the top right

## Setup Virtual Environment
- First ensure you are in the directory of this repo
- python -m venv venv
- ./venv/Scripts/activate
- pip install -r requirements.txt
- Then when running main.ipynb, select venv as the kernel (in the top right)
